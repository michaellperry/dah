With Jinaga, you record historical facts as JSON documents. Express a fact that represents you as a user. You want your fact to be unique, so you should use a random number. I've generated one for you.* 

var user = {
    type: 'Jinaga.User',
    identity: -7164293
};
j.fact(user);

* If you are pairing with a friend, be sure not to use your friend's number!
